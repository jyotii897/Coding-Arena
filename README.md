# Coding Arena

Coding Arena is a web platform where users can solve programming problems and see real-time feedback.

## Features

- Solve problems in multiple programming languages
- Real-time input/output interface
- Leaderboard and scoring
- User authentication

1. Clone the repository  
2. Install dependencies in both `client` and `server` folders with `npm install`
3. Start the backend server and frontend client
4. Visit `localhost:3000` (or your deployed link)
